Variables - Storing Data

Variables are containers for storing data values.

Three Ways to Declare Variables:

Declaration:

// let - can be changed let leaves = 100; leaves = 150; // OK! // const - cannot be changed const SEASON = "Fall"; // SEASON = "Winter"; // ERROR! // var - old way (avoid) var temperature = 65;

Best Practices:

Use let when value will change

Use const when value won't change

Avoid var in modern code

Variable Naming Rules:

  • Must start with letter, $, or _
  • Can contain letters, numbers, $, _
  • Case sensitive (apple ‚â† Apple)
  • Can't use reserved words (let, if, for)

Order of Operations (PEMDAS)

JavaScript follows mathematical order:

Priority Operation Symbol Example
1 Parentheses ( ) (2 + 3) * 4 = 20
2 Exponents ** 2 ** 3 * 2 = 16
3 Multiplication
Division		 * / 10 + 3 * 2 = 16
4 Addition
Subtraction		 + - 10 - 3 + 2 = 9

Examples:

let result1 = 10 + 5 * 2; // 20 (not 30!) let result2 = (10 + 5) * 2; // 30 let result3 = 100 / 10 * 5; // 50 (left to right) let result4 = 2 ** 3 ** 2; // 512 (right to left for **) let result5 = 10 % 3 + 2; // 3 (remainder then add)

Tip: When in doubt, use parentheses to make your intention clear!

Break and Continue

Control loop execution with break and continue.

Break - Exit Loop:

// Find first even number for (let i = 1; i <= 10; i++) { if (i % 2 === 0) { console.log(`Found even: ${i}`); break; // Exit loop } } // Output: Found even: 2 // Search in array let fruits = ["apple", "pear", "grape"]; for (let fruit of fruits) { if (fruit === "pear") { console.log("Found pear!"); break; } }

Continue - Skip Iteration:

// Skip odd numbers for (let i = 1; i <= 5; i++) { if (i % 2 === 1) { continue; // Skip odd } console.log(`Even: ${i}`); } // Output: // Even: 2 // Even: 4 // Skip specific value let temps = [65, 70, -999, 75]; for (let temp of temps) { if (temp === -999) continue; console.log(`Temp: ${temp}¬∞F`); }

break = Exit entire loop | continue = Skip to next iteration
